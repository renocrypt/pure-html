Pure HTML Project

A playground for modern web development without bundlers

About This Project

This project demonstrates how to use modern web technologies directly in the browser without relying on bundlers or complex build tools.

Explore the examples to see what's possible with pure HTML, CSS, JavaScript, and TypeScript.
