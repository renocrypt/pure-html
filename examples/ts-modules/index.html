About This Example

This example demonstrates how to use TypeScript directly in the browser without using a bundler or transpiler. We're using import maps and the browser's native ES module support.

Open your browser's developer console to see the TypeScript code in action.

    How It Works

    We're using an import map to tell the browser where to find TypeScript modules. Then we load TypeScript files with a .js extension, which the browser can execute directly.

    Key Files:

    • main.js - The entry point with TypeScript syntax
    • todo.js - The Todo class with TypeScript interfaces
    • store.js - Storage utilities with TypeScript generics

    Code Example:

    // TypeScript interfaces
interface Todo {
  id: number;
  text: string;
  completed: boolean;
}

// Type-safe functions
function createTodo(text: string): Todo {
  return {
    id: Date.now(),
    text,
    completed: false
  };
}

// Type annotations used directly in the browser!
const todos: Todo[] = [];

    The browser ignores the TypeScript type annotations at runtime but executes the JavaScript parts of the code.
